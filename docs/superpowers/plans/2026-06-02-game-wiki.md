# 游戏百科功能 Implementation Plan

> 新增游戏百科功能模块，覆盖 Squad 的武器/兵种/载具/地图/阵营/模式，共 7 个路由页面 + 数据层 + 2 个公共组件。

**Architecture:** 所有百科数据统一在 `src/data/wiki/` 中管理（6 个分类数据文件 + 1 个索引），页面通过 `WikiLayout` + `WikiSidebar` 两个通用组件实现文档式布局，百科大厅用卡片网格入口。

**Tech Stack:** React 18 + TypeScript + React Router + Tailwind CSS + lucide-react

## Tasks

### Task 1: 百科数据层 - 统一导出与索引
创建 `src/data/wiki/index.ts`，导出所有数据模块、统计数据、分类配置。

### Task 2: 武器数据
创建 `src/data/wiki/weapons.ts`，~15 条武器条目。

### Task 3: 兵种数据
创建 `src/data/wiki/classes.ts`，~10 条兵种条目。

### Task 4: 载具数据
创建 `src/data/wiki/vehicles.ts`，~12 条载具条目。

### Task 5: 地图/阵营/模式数据
创建 `src/data/wiki/maps.ts`（~26条）、`factions.ts`（4条）、`modes.ts`（4条）。

### Task 6: WikiSidebar 公共组件
创建 `src/components/WikiSidebar.tsx`，支持分组显示和移动端折叠。

### Task 7: WikiLayout 公共组件
创建 `src/components/WikiLayout.tsx`，左侧边栏 + 右侧内容区布局。

### Task 8: 百科大厅
创建 `src/pages/Wiki.tsx`，6 张卡片网格入口。

### Task 9-13: 子页面
创建 Weapons.tsx, Classes.tsx, Vehicles.tsx, Maps.tsx, Factions.tsx, Modes.tsx。

### Task 14: 路由注册
在 `src/App.tsx` 新增 7 条路由。

### Task 15: 导航更新
在 `src/pages/Home.tsx` 的 Nav 和 Footer 中追加「百科」链接。

### Task 16: 最终验证
类型检查 + Lint + 构建 + 预览验证。
