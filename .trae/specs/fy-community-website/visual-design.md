# 肥鸭战队（FY）官网社区视觉设计规范

## 概述

本规范定义了肥鸭战队（FY）官网社区的完整视觉系统，涵盖色彩、字体、LOGO、吉祥物、间距、阴影、图标等核心设计要素。整体风格定位为**现代电竞风 + 活泼可爱元素**。

## 1. 色彩系统

### 1.1 主色（黄色系）

| 色名 | 色值 | 用途说明 |
|------|------|----------|
| 肥鸭亮黄 | #FFE14D | 最高强调色，主按钮、核心图标 |
| 肥鸭金黄 | #FFC107 | 主品牌色，LOGO、导航激活状态 |
| 肥鸭橙黄 | #FF9800 | 次级强调色，渐变过渡、成就等级 |
| 肥鸭暖黄 | #FFB74D | 辅助装饰色，卡片边框、标签背景 |
| 肥鸭深黄 | #F57C00 | 深色强调色，文字阴影、渐变暗部 |

### 1.2 辅助色

| 色名 | 色值 | 用途说明 |
|------|------|----------|
| 电竞深黑 | #0A0A0F | 主背景色 |
| 暗夜灰 | #14141B | 卡片背景色 |
| 石墨灰 | #1E1E2A | 悬浮面板、下拉菜单 |
| 钢灰 | #2A2A3A | 边框、分割线 |
| 银灰 | #8A8A9A | 次级文字 |
| 月白 | #E8E8F0 | 主要正文文字颜色 |

### 1.3 强调色

| 电光蓝 | #00D4FF | 链接色、信息提示 |
| 霓虹紫 | #B829F7 | 特殊活动、VIP标识 |
| 烈焰红 | #FF3D57 | 错误状态、删除操作 |
| 生机绿 | #00E676 | 成功状态、在线指示 |

## 2. LOGO 设计规范

肥鸭战队LOGO由**肥鸭形象**与 **FY 字母组合**构成。

## 3. 吉祥物形象设计

吉祥物名为**"肥肥"**（Fatty），是一只圆滚滚的黄色小鸭子。

## 4. 字体排版规范

| 用途 | 字体 |
|------|------|
| 标题字体 | 得意黑（Smiley Sans） |
| 正文字体 | 思源黑体（Source Han Sans） |
| 数字/数据 | DIN Alternate / Roboto Mono |
| 英文标题 | Inter / Montserrat |

## 5. 间距系统

基于 4px 基准单位：space-1(4px) 到 space-20(80px)。

## 6. 阴影系统

shadow-sm, shadow-md, shadow-lg, shadow-xl, shadow-glow 共5级。

## 7. 图标风格规范

线性图标（Outline Style），2px 描边，端点圆角。

## 8. 组件视觉规范

主按钮、次级按钮、卡片、输入框、标签等核心组件规范。

## 9. 视觉风格原则

深色沉浸式背景 + 高对比度文字 + 霓虹光效点缀 + 活泼可爱元素。

*本规范由 UI/UX 设计团队制定。*
